Textural Properties¶

texturalproperties.get_entropy(image)[source]¶

Returns the number of vertical and horizontal edges in image

Parameters:
  • image (array_like) – input image
  • threshold_ratio (float, optional) – Ratio between max and min threshold. Default value is 0.33. Mutual use with min and max threshold
  • min_threshold (float, optional) – min value for edge detection. Default is None.
  • max_threshold (float, optional) – max value for edge detection. Default is None.
Returns:

out –

  • vertical edges
    : int
    number of vertical edges
  • horizontal edges
    : int
    number of vertical edges
Return type:

tuple
